📊 Online Courses Analytics Dashboard (Power BI)
📌 Project Overview

This project analyzes an online courses dataset using Microsoft Power BI. The objective is to uncover meaningful insights about course categories, learner engagement, instructor performance, language distribution, and course popularity through an interactive dashboard.

The project demonstrates the complete Business Intelligence workflow:

Data Cleaning using Power Query
Data Modeling
DAX Measures
Interactive Dashboard Design
Business Insights & Recommendations
📁 Dataset

The dataset contains information about online courses, including:

Course Title
Category
Sub-Category
Rating
Number of Viewers
Language
Instructor
Course Type
Duration
Subtitle Availability
🎯 Project Objectives
Analyze the distribution of online courses.
Compare average course ratings across categories.
Analyze learner engagement using course views.
Identify the most popular course categories.
Understand language distribution.
Analyze subtitle availability.
Find top-rated instructors.
Study the relationship between course duration and popularity.

💡 Key Insights
Data Science and Information Technology contain the largest number of courses.
Average ratings remain consistently high across most categories.
English is the dominant language in the dataset.
Viewer engagement varies significantly across categories.
Most cleaned records include subtitle support.
Longer courses do not always receive more viewers.
Highly rated instructors appear repeatedly among top-performing categories.
📌 Business Recommendations
Expand Data Science, AI, and Cloud Computing courses.
Continue offering subtitle support for global accessibility.
Promote highly rated instructors.
Improve course quality rather than simply increasing duration.
Increase marketing efforts for low-engagement categories.
